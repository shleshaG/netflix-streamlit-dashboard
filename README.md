# Netflix Data Analysis Dashboard

This project is an interactive dashboard built using Python and Streamlit
to analyze Netflix movies and TV shows data.

## Features
- Movies vs TV Shows analysis
- Year-wise content trends
- Genre analysis
- Interactive filters

## Tech Stack
- Python
- Pandas
- Streamlit
- Matplotlib
- Seaborn

## Dataset
Netflix Movies and TV Shows dataset from Kaggle.

## How to Run
streamlit run app.py
